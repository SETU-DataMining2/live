Setup

The setup is essentially the same as other labs, but with two exceptions:

  • Have helper script that stores plotting functions. The function are simple wrapper around commands that we use to date, but will cut down on pimping time.
  • Save models for later use. This is something that I should have been using before now and might be of use for you in the Kaggle competition assignment.

To start the lab:

  • Create folder (you prob/should have it already)BC_Wisconsin.
  • Download and save in folder the script plot_lib.py.
  • Create notebook with level 1 heading "xAI - BC_Wisconsin", and level 2 heading "Setup".
