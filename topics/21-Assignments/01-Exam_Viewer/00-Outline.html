CA1 - Exam Viewer

Aim of Assignment

Develop a StudyClix-like website to help students prepare for Discrete Mathematics exams. Ideally the website/dashboard should:

  • Allow viewing/searching of multiple exam papers.

  • Filter by topic, by keyword, by difficulty, by ...

    • Shows each of the questions that matches the selection criteria and any other helpful information.

The following are screenshots of my attempt at this, to help give you context. Your job is not to just reproduce what I did but take the idea and see how far you can push it.

Implementation

In contrast to other assignments I want to give you maximum freedom to decide how carry out this implementation.

At a minimum you need some way to generate a dashboard and parse PDFs. For these two subtasks, I used the following but feel free to use alternatives:

  • Website is a dashboard based on streamlit but other options are gradio, or niceGUI.

  • Parsing of PDF documents using pymupdf. There are many other python based PDF parsers so you might find a more suitable option.

I'm intentionally leaving awful lot out — such as how to preprocess exams papers — this is all up to you.

Deliverable

This assignment will be managed using github (instead of Moodle) so I can see weekly progress.
